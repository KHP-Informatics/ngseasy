# mHMM

## m-HMM

This is the R software for the m-HMM, the mixture hidden Markov model used to detect copy number variation using next generation sequencing data.

- https://www.stt.msu.edu/users/hengwang/mHMM.html 

### Requirements:
R 3.0.0 or later.
